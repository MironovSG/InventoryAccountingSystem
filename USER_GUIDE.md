# Руководство пользователя - Система учета ТМЦ

## Для специалистов (Пользователей)

### Вход в систему

1. Откройте браузер и перейдите по адресу: `http://localhost:8080`
2. Введите свой логин и пароль
3. Нажмите кнопку **"Войти"**
4. Вы будете автоматически перенаправлены на главную страницу

### Главная страница

После входа вы увидите главный экран с меню слева и быстрым доступом к функциям:

#### Левое меню содержит:
-  **Главная** - главная страница с обзором
-  **Каталог ТМЦ** - просмотр доступных материалов
-  **Создать заявку** - создание новой заявки на выдачу ТМЦ
-  **Мои заявки** - просмотр ваших заявок
-  **История заявок** - общая история всех заявок

#### На главной странице отображается:
- Количество ваших заявок
- Последние уведомления
- Последние созданные заявки

### 1. Просмотр каталога ТМЦ

#### Как открыть:
Нажмите **"Каталог ТМЦ"** в левом меню

#### Возможности:
✅ Просмотр всех доступных материалов (только чтение, без редактирования)
✅ Фильтрация по любому столбцу:
- **По складу**: Расходные материалы / Запасные части / Комплектующие
- **По категории**: выбор из списка категорий
- **По уровню запасов**: В наличии / Мало / Отсутствует
- **По коду материала**: фильтр по коду
- **По названию**: фильтр по названию

#### Таблица каталога показывает:
- Код материала
- Название
- Категория
- Склад (где хранится)
- Расположение на складе
- Остаток (количество)
- Единица измерения
- Статус (зеленый/желтый/красный индикатор)

### 2. Создание заявки на выдачу ТМЦ

#### Как создать заявку:

**Способ 1:** Нажмите  **"Создать заявку"** в левом меню

**Способ 2:** На главной странице нажмите на карточку "Создать заявку"

#### Окно создания заявки:

Откроется модальное окно с таблицей из **4 строк** и **4 столбцов**:

| Код материала | Название | Количество | Наименование работ |
|---------------|----------|------------|-------------------|
| (ввод)        | (ввод)   | (ввод)     | (ввод)            |
| (ввод)        | (ввод)   | (ввод)     | (ввод)            |
| (ввод)        | (ввод)   | (ввод)     | (ввод)            |
| (ввод)        | (ввод)   | (ввод)     | (ввод)            |

#### Заполнение заявки:

1. **Код материала** - введите код материала из каталога
2. **Название** - введите название материала
3. **Количество** - укажите нужное количество
4. **Наименование работ** - опишите для каких работ нужны материалы

**Примечания:**
- Первая строка обязательна для заполнения
- Остальные строки заполняются по необходимости
- Можно добавить дополнительные строки кнопкой **"+ Добавить строку"**
- Код материала должен существовать в каталоге ТМЦ

#### Отправка заявки:

**Две кнопки внизу:**

- 🟢 **"Отправить в работу"** - отправить заявку МОЛ
  - После отправки МОЛ получит push-уведомление о новой заявке
  - Вы увидите подтверждение об успешной отправке
  
- ⚪ **"Отменить"** - отменить создание заявки и закрыть окно

### 3. Просмотр своих заявок

#### Как открыть:
Нажмите **"Мои заявки"** в левом меню

#### Что вы увидите:

Таблица со всеми вашими заявками:

| Номер заявки | Дата заявки | Статус | Наименование работ | Действия |
|--------------|-------------|--------|-------------------|----------|
| REQ-... | 12.01.2026 | Открыта | Ремонт... |  Просмотр |

#### Статусы заявок:

- **Открыта** (🔵) - заявка создана, ожидает обработки
- **В работе** (🟡) - МОЛ принял заявку в работу
- **Закрыта** (🟢) - материалы выданы, заявка выполнена
- **Отклонена** (🔴) - заявка отклонена МОЛ

#### Просмотр деталей заявки:

Нажмите кнопку **" Просмотр"** чтобы увидеть:
- Полную информацию о заявке
- Список всех запрошенных материалов
- Количество запрошенное, одобренное и выданное
- Дату создания и выдачи
- Причину отклонения (если заявка отклонена)

**Примечание:** Это окно только для просмотра, редактирование недоступно.

### 4. Просмотр истории заявок

#### Как открыть:
Нажмите **"История заявок"** в левом меню

#### Что отображается:

Полная история **всех заявок** всех пользователей системы.

#### Таблица истории:

| Номер заявки | Дата | Специалист | Группа | Работы | Статус | Дата выдачи |
|--------------|------|------------|--------|--------|--------|-------------|
| REQ-... | 12.01 | Иванов И.И. | АРМ | Ремонт | Выдана | 13.01 |

#### Фильтры:

Вы можете фильтровать историю по:
- **Дате** (с ... по ...)
- **Рабочей группе** (АРМ / ГОТО / ГОКС)
- **Статусу заявки**

Кнопки:
- **"Применить"** - применить фильтры
- **"Сбросить"** - сбросить все фильтры

### 5. Уведомления

#### Как работают уведомления:

Вы получаете **автоматические push-уведомления** когда:
-  МОЛ принял вашу заявку в работу
-  Материалы по вашей заявке выданы
-  Ваша заявка отклонена (с указанием причины)

#### Просмотр уведомлений:

**На главной странице:**
- Видны последние 5 уведомлений
- Красный бадж показывает количество непрочитанных

**На странице уведомлений:**
- Нажмите на бадж с количеством уведомлений
- Увидите все уведомления
- Непрочитанные выделены синим цветом
- Нажмите на уведомление, чтобы отметить прочитанным

**Функции:**
- ✓ **"Отметить все как прочитанные"** - массовая отметка
- **"Непрочитанные"** - показать только новые
- **"Все"** - показать все уведомления

### 6. Типичные сценарии использования

#### Сценарий 1: Запрос материалов для ремонта

1. Зайдите в **"Каталог ТМЦ"**
2. Найдите нужные материалы и запомните коды
3. Нажмите **"Создать заявку"**
4. Заполните таблицу:
   - Код: `RES-100K`
   - Название: `Резистор 100кОм`
   - Количество: `10`
   - Работы: `Ремонт платы клиента №12345`
5. Нажмите **"Отправить в работу"**
6. Дождитесь уведомления о статусе заявки

#### Сценарий 2: Проверка статуса заявки

1. Зайдите в **"Мои заявки"**
2. Найдите нужную заявку в списке
3. Посмотрите текущий статус
4. Нажмите **"Просмотр"** для деталей
5. Если статус "Выдана" - можете забрать материалы у МОЛ

#### Сценарий 3: Просмотр истории выдач

1. Зайдите в **"История заявок"**
2. Установите фильтр по датам (например, последний месяц)
3. Выберите свою рабочую группу
4. Нажмите **"Применить"**
5. Просмотрите историю всех выдач

### 7. Часто задаваемые вопросы

**В: Могу ли я редактировать каталог материалов?**
О: Нет, каталог доступен только для просмотра. Редактирование может выполнять только МОЛ.

**В: Могу ли я отменить отправленную заявку?**
О: Нет, после отправки заявки её отменить нельзя. Свяжитесь с МОЛ напрямую если нужна отмена.

**В: Как долго обрабатывается заявка?**
О: Это зависит от загруженности МОЛ и наличия материалов. Вы получите уведомление при изменении статуса.

**В: Что делать если материала нет в каталоге?**
О: Свяжитесь с МОЛ для добавления материала в каталог.

**В: Сколько материалов можно запросить в одной заявке?**
О: По умолчанию 4 позиции, но можно добавить больше строк кнопкой "+ Добавить строку".

**В: Получу ли я уведомление когда материалы готовы к выдаче?**
О: Да, вы получите push-уведомление когда статус изменится на "Выдана".

**В: Где посмотреть причину отклонения заявки?**
О: Зайдите в "Мои заявки", нажмите "Просмотр" на отклоненной заявке - увидите причину внизу окна.

### 8. Рабочие группы

Система поддерживает три рабочие группы:

- **АРМ** - Автоматизированное рабочее место
- **ГОТО** - Группа оперативно-технического обслуживания
- **ГОКС** - Группа оперативно-контрольных систем

Ваша группа отображается в профиле и в истории заявок.

### 9. Советы по эффективной работе

✅ **Проверяйте каталог** перед созданием заявки - убедитесь что материал есть в наличии

✅ **Указывайте точное количество** - это ускорит обработку заявки

✅ **Подробно описывайте работы** - МОЛ сможет лучше понять вашу потребность

✅ **Проверяйте уведомления регулярно** - чтобы не пропустить готовность материалов

✅ **Используйте фильтры в каталоге** - это экономит время при поиске материалов

✅ **Создавайте заявки заранее** - не ждите последнего момента
